    # 哪吒之魔童降世

    《哪吒之魔童降世》于2019年7月26日在中国内地上映，一经上映便受到多方关注，票房更是节节高升，截止到2019年10月10日，以49.7亿元的票房成绩在总票房排行榜上位列第二，仅次于《战狼2》。很喜欢动漫的我也去观看了这部影片，因而写下这篇文章记录影片相关信息。

    我和室友是在2019年7月28日晚上去影院看的《哪吒之魔童降世》，个人很喜欢这部动漫电影，里面胖胖的太乙真人，生而为魔的丑萌哪吒，背负龙族命运的龙太子，还有殷夫人、李靖、申公豹等角色丰富个性鲜明的人物。故事中有笑点也有泪点，让人跟着哪吒的命运而跌宕起伏，看的过瘾。

    # 影片歌曲

    • 主题曲：《哪吒》 - GAI&大痒痒
    • 片尾曲：《今后我与自己流浪》 - 张碧晨

    # 经典语录

    影片中的很多对话、语录都极富教育意义，发人深省，听的让人热血沸腾。这里便总结了部分哪吒中的经典语录，待人生失意、颓废时，来翻看一下，想必就能找到继续奋进前行的动力。

    • 如果你问我，人能否改变自己的命运，我也不晓得，但是不认命，就是哪吒的命。
    • 人心中的成见是一座大山，任你怎么努力都休想搬动
    • 若命运不公，就和它斗到底。
    • 他们把我当妖怪，我就当妖怪给他们瞧瞧。
    • 今天是我的生日宴，你们谁都不准哭。
    • 我们一生中，来来往往会遇见很多人，但我们都在不自知的等待一个能看见自己的人。
    • 一生中能改变命运的机会可不多呀。
    • 我自己的命自己扛，不连累别人。
    • 别人的看法都是狗屁，你是谁只有你自己说了才算，这是爹教我的道理。
    • 别挣扎了，你生来就是魔丸这是命中注定。
    • 去你个鸟命，我命由我不由天，是魔是仙我自己说了才算。
    • 生而为魔，那又如何。
    • 我是小妖怪，逍遥又自在，杀人不眨眼，吃人不放盐，一口七八个，肚皮要撑破，茅房去拉屎，想起忘带纸。
    • 生活你全是泪，没死就得活受罪，越是折腾越倒霉，越有追求越悲催，垂死挣扎你累不累，不如瘫在床上睡。
    • 关在府里无事干，翻墙捣瓦摔瓶罐。来来回回千百遍，小爷也是很疲倦。
    • “想当初你还是娘肚子里的一颗红烧狮子头，现在你都长成有鼻子有眼的帅小伙了。”
    • “我只是沙子进了眼睛…”
    • “哪吒是你什么人？” “他是我儿”
    • “吒儿，这是父亲送你的平安符，能保佑你平安长大，长命百岁。”
    • “你是我唯一的朋友” “你也是我唯一的朋友”
    • “白白搭上一条命，你傻不傻” “不傻，谁和你做朋友”
    • “爹一直对你很严，知道你心里有气，别在意别人的看法，你是谁，只有你自己说了算，永远不要放弃。”

    # 人生感悟

    我们每个人生来便负重前行，只是幼时无知，待渐渐长大，经历世事才明白，生不是由我决定的，但命运也许是自己掌控的。

    影片中的哪吒，刚出生时因魔丸而生性暴戾，因魔丸而不容于世，生来便背负这世间最大的恶意，受母亲教化压下魔性，露出笑容，彼时的哪吒是善良的，是渴望被爱的。只是生活从来不是如此平静无波，否则也不会有今天我们所看到的《哪吒》了。百姓的惧怕、世人的偏见、父母的禁锢使哪吒看见了这世间对自己的排斥，明明想和城中的小孩一起玩耍却被同龄的伙伴仍石头，明明在捉妖，可因为人们只看见坏的一面而被讨伐，就如同我们努力的想要生存、生活的更好，却总是被现实打击一样。

    庆幸的是影片里的哪吒有一个好师傅，有一对爱他胜过自己父母，成功的感化他，成功的让哪吒明白“我命由我不由天，是魔是仙我自己说了才算”。很多人都经历过低谷，然后又站了起来，也许是朋友、是父母让自己决定在努力一把，也许是自己想要活出精彩不被人看扁。纵前方天雷滚滚，又有何理由后退，岂不是连3岁的小孩子都不如了。
