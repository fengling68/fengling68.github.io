# 第一期：故宫博物院

一百八十六万余件馆藏

# 千里江山图

  • 国宝守护人：李晨
  • 北宋徽宗年间18岁少年王希孟
  • 青绿山水画
  • 绝世之画作，天才之少年。黑墨勾山石，青绿施重彩。咫尺有千里，细看有生趣。
  • 宋朝版航拍中国
  • 北宋徽宗自创瘦金体书法
  • 踏花归来马蹄香
  • 太师蔡京题跋
  • 十三年后，靖康乱，徽宗被囚，蔡京被流放。
  • 矿物质颜色
    • 水墨打底
    • 红色赭石
    • 石绿绿松石或孔雀石
    • 绿
    • 青色

# 瓷母

  • 国宝守护人：王凯
  • 乾隆令景德镇工匠制作
  • 宋代到清代十七种最具代表性的釉彩
  • 景德镇督陶官唐英
  • 王羲之右军先生
  • 黄公望一峰先生

# 石鼓

  • 国宝守护人：梁家辉
  • 中华第一古物，共十面。
  • 大篆向小篆过渡的一种文字，石鼓文。
  • 汉字演进历史上的活化石
  • 民间推举中国九大振国之宝之一
  • 司马光和其父司马池寻找石鼓
  • 司马光《资治通鉴》
  • 古物南迁
  • 二战文物保护传奇一件未失
  • 七年时间核对文物：梁金生
  • 故宫守护队队歌

# 第二期：湖北省博物馆

湖北省博物馆24万余件馆藏

# 越王勾践剑

  • 国宝守护人：段奕宏
  • 1965年12月发掘
  • 天下第一剑
  • 千年不朽
  • 春秋末年，越国勾践战败，卧薪尝胆。
  • 范蠡，扶同大夫。
  • 树枝晶的金相组织，材料好，环境好，带剑鞘，千年不腐。
  • 吴王夫差矛

# 云梦睡虎地秦简

  • 国宝守护人：撒贝宁
  • 1975年12月湖北省云梦县睡虎地秦代墓葬
  • 1155枚竹简四万多字
  • 墓葬主人“喜”秦国基层官吏
  • 千年竹简，见证国家统一从篆书到隶书的转变，秦隶。
  • 干千年，湿万年，不干不湿只半年。
  • 《封诊式》环境保护，刑侦，户籍管理。
  • 二十世纪知名考古学家：陈振裕

# 曾侯乙编钟

  • 国宝守护人：王刚
  • 1978年在湖北随县曾侯乙墓出土
  • 六十五件，迄今发现的先秦编钟里规模最大，数量最多，铸造最精，音乐性能最好，保存最完整的一套编钟，改写了世界音乐史。
  • 战国初期，楚惠王&曾侯乙
  • 为编钟定音的均钟木(校音器)
  • 第一次敲响：出土三月后国庆节
  • 第二次敲响：1984庆祝中国成立三十周年
  • 第三次敲响：1997年香港回国
  • 武汉音乐学院谭军，
  • 武汉音乐学院青年编钟乐团。
  • 《交响曲1997—天、地、人》

# 第三期：河南博物院

河南博物馆十七万余件

# 妇好鴞尊

  • 国宝守护人：刘涛
  • 妇好，商王武丁的妻子，大祭司，军事家三千多年前的商朝酒樽
  • 鴞：猫头鹰河南安阳殷墟妇好墓，青铜钺，“妇好”铭文。
  • 敌族羌方
  • 妇好鴞尊一对，一件河南博物馆，一件中国国家博物馆。
  • 第一位有文字记载的女英雄
  • 满满的墓葬陪葬品
  • 考古研究所研究员：郑振香
  • 唐际根
  • 三代中国考古学者

# 贾湖骨笛

  • 国宝守护人：蔡国庆&蔡轩正
  • 千年古乐传承不灭
  • 舞阳县贾湖遗址
  • 鹤类尺骨制成的七阶乐器
  • 是迄今中国发现的时代最早、保存最完整的乐器，其出土改写了中国和世界音乐的历史。
  • 出土了四十多只，八千多年前七阶音阶。
  • 民族音乐研究人：萧兴华、杜佳慧
  • 江上清风游
  • 河姆渡遗址，浙江省余姚市河姆渡镇

# 云纹铜禁

  • 国宝守护人：雷佳音
  • 河南博物馆九大镇院之一
  • 1978年河南淅川下寺二号春秋中期楚墓出土
  • 楚庄王之子，子庚之墓
  • 子庚，一国之相
  • 体积最大，年龄最大，工艺最美
  • 《酒诰》出自《尚书·周书》劝人不要酿酒酗酒，是中国最早的禁酒令。
  • 摆酒的案子称为“禁”，也是祭祀祖先之物。
  • 楚庄王：一鸣惊人，问鼎中原，饮马黄河，春秋五霸之一。
  • 失蜡法：古代金属制造的一种方法。
  • 以史为鉴，莫贪杯，多做好事。
  • 中国航发高级工程师—吴庆辉
  • 飞机涡轮叶片，铸造和锻造。
  • 中国，美国，英国，俄罗斯自主研发飞机发动机的四个国家。

# 第四期：陕西历史博物馆

陕西历史博物馆一百七十万余件馆藏

# 葡萄花鸟纹银香囊

  • 国宝守护人：马苏
  • 1970出土于何家村窑藏
  • 子母口
  • 唐玄宗年间
  • 长恨歌，杨贵妃，唐明皇，马嵬坡，梨园戏蝶，霓裳羽衣曲。
  • 肌肤已坏，而香囊犹在。
  • 天生丽质难自弃，长恨一曲千古迷。
  • 香气四溢绕悬梁，梦回千年见盛唐。
  • 葡萄，西域传入中国。
  • 此物是古丝路文明共荣的缩影。
  • 北大考古文博学院丝绸之路考古负责人：齐东方
  • 何家村遗宝，千余件宝物。

# 杜虎符

  • 国宝守护人：刘奕君
  • 忠诚、信任、责任
  • 秦国兵甲之符，调兵遣将。
  • 现存秦虎符仅三枚
  • 右在君，左在杜。秦唯一的君，秦惠文王。
  • 密码研究学者刘胜利博士、密码学王小云。
  • 密码学，认证&保密。
  • 鱼符，调兵遣将，官员象征，合同、金龟婿来历。
  • 国家密码管理局，SM算法系列，自主研发。

# 懿德太子墓壁画《阙楼仪仗图》

  • 国宝守护人：郭涛
  • 1971年，乾县唐懿德太子李重润墓出土。
  • 一座母阙，两座子阙，『三出阙』建筑形象。
  • 左青龙右白虎
  • 三军仪仗：步兵，骑兵，车马队，一百九十六人。
  • 政治，经济，文化，商贸，东市，西市。
  • 阙同缺，路过阙楼，想想自己有什么缺点。
  • 壁画修复师团队
  • 张锦秋老师，一辈子，一座城，大唐气象建筑师，陕西历史博物馆设计人。2002年设计建大唐芙蓉园，天策府原型。大明宫复原设计，上海世博会展览，以小见大。
  • PS：《唐朝穿越指南》《唐朝定居指南》两本现代书。

# 第五期：辽宁省博物馆

辽宁省博物馆 馆藏十二万余件 升则飞腾于宇宙之间，隐则潜伏于波涛之内。

# 宋人摹顾恺之《洛神赋图》卷

  • 国宝守护人：陈晓
  • 《洛神赋图》东晋时期作者顾恺之，字长康，被称为“画圣”，也有说吴道子是画圣。
  • 《洛神赋》词赋作者是三国时期文学家曹植，主要讲曹植和洛水女神相见相爱到离别的爱情故事。
  • 顾恺之，倒吃甘蔗，渐入佳境。才绝，画绝，痴绝。传神写照，正在阿堵之中。
  • 洛神，宓妃，伏羲女娲之女。因迷恋洛河两岸景色，降临人间，来到洛阳。
  • 仙才曹植，字子建。
  • 思卿不见，难赴黄泉。
  • 《仙才叹》仙是洛神，才是子建，叹是顾恺之。
  • 人生八苦，生老病死爱别离，恨长久，求不得，放不下。
  • 迁想妙得，以形写神，顾恺之的两个观点影响了中国绘画历史。
  • 中国美术学院，叶露盈，绘本版《洛神赋图》以线造型，书法入画，是中国画的传统。根据自己的想像，去创造无限的可能。
  • 《山海经》《抱朴子》

# 铜鎏金木芯马镫

  • 国宝守护人：关晓彤
  • 北燕冯素弗墓，四世纪初期只有一个，五世纪发展成两个。
  • 改变了中国骑兵史，乃至世界的骑兵史。
  • 马镫还被叫做中国靴子。
  • 奥运会马术运动员，华天，中国第一骑士。
  • 华天的一匹马叫做“花木兰”。其他马的名字如“孙悟空”、“武松”、“乾隆”。
  • 伊顿公学，三个80后荣誉学生：华天，威廉王子，哈里斯王子。

# 唐摹王羲之一门书翰卷(万岁通天贴)

  • 国宝守护人：宁静
  • 王羲之真作没有，临摹的都是下真迹一等作品。
  • 第一幅《姨母帖》
  • 集王羲之，王献之等王氏七人的书法摹本，世间仅此一本。
  • 王羲之真迹《丧乱帖》，现藏于日本宫内厅三之丸尚藏馆。
  • 公元697年，一代女皇武则天，从王綝，字方庆的手里得到了王氏一门书法真迹。
  • 唐太宗李世民，立遗嘱要《兰亭序》陪葬。
  • 双钩填墨的摹本最接近真迹，故历代评书者皆称这些摹本为“下真迹一等”。
  • 狄仁杰、王方庆
  • 狄仁杰死后，武则天痛哭“朝堂空矣”。
  • 《国宝沉浮录》
  • 1922年，溥仪偷运古籍古董。1945年，伪满国失败，伪军趁虚而入，损毁国宝。
  • 杨仁凯先生
  • 文物身份证，文物普查。
    • 第一次九百年前，宋徽宗时期；
    • 第二次三百年前，清朝嘉庆年间；
    • 第三次上世纪八十年代，杨仁凯先生和其他人普查；
  • 《中国古代书画鉴定笔记》
  • 谢稚柳、启功，徐邦达、杨仁凯、刘久庵、傅熹年、谢辰生。
  • 中国古代书画巡回鉴定组，鉴千古书画，正中华文脉。

# 第六期：湖南省博物馆

湖南省博物馆

# 长沙窑青釉褐乳诗文执壶

  • 国宝守护人：何炅
  • 诗情画意，国色天香。
  • 所题诗歌“春水春池满，春时春草生，春人饮春酒，春鸟啼春声。”体现了长沙窑的诗文装饰特征。
  • 唐朝的酒壶，一千两百年前。
  • 长沙窑，釉下多彩，打破了以青、白为主的生产格局。形成了南青、北白、长沙彩的鼎力之势。经由海上丝路，远销海外占领全球。
  • 邢窑白瓷：唐五代时期著名北方窑瓷。
  • 越窑青瓷：中国古代最著名的青瓷窑系，始创于东汉时期，最早在越窑烧制，因此也被称为“母亲瓷”。
  • 公元九世纪，长沙窑扩张，远销海外。外销瓷的开始。
  • 文化使者，海上丝绸之路。
  • 印度尼西亚，打捞了一艘唐代的阿拉伯商人沉船“黑石号”，六万七千多件瓷器。
  • 绘画艺术装饰瓷器。重走海上丝绸之路的人，林安。

# 辛追墓T型帛画

  • 国宝守护人：雷佳
  • 颜色历经两千多年，依旧鲜艳如初，不可思议。
  • 1972年出土于长沙马王堆一号汉墓锦饰内棺盖板上，长约两米，上下末端分别宽约九十和五十厘米。
  • 毛笔绘制于丝帛之上，颜色富丽，线条流畅，是中国古代美术史教材必会收录的绘画佳作。
  • 画主人文明世界，千年不朽的辛追。西汉文帝年间，长沙国丞相利苍的夫人辛追，马王堆一号汉墓主人，出土时尸形完整，全身润泽，是世界上保存最好的千年湿尸，丰富的随葬品也是研究汉代风貌的重要实物。
  • 红颜只为夫君装
  • 两千年，一瞬间，永生之梦光华再现。
  • 上方日月，神灵，异兽，天国想像，中间驻杖而立的老妪，认为是辛追，下方代表地下。
  • 出葬时的导引，下葬时覆盖内棺之上，叫铭旌，也就是幡。
  • 第一层木炭，第二层白膏泥，紧密的空间。
  • 《月亮粑粑》童谣
  • 太阳黑子，一克丝线可拉至900米长。
  • 内容丰富，繁而不乱，层次分明，有主有次动静结合，现实和虚幻也结合的很好，主题又突出。展现了汉代高超的绘画艺术和汉代的哲学思想，包括当时的宇宙观、生命观的完美结合。
  • 《辛追传奇》片尾曲

# 皿方罍

  • 国宝守护人：黄渤，王嘉
  • 皿方罍(lei)，酒器礼器。
  • 1919年湖南省桃源县农民意外发现，后经买卖流失海外，盖身分离，近一个世纪后才归湘。
  • 八十八厘米，是晚商青铜器鼎盛期代表之作。『皿而全作父己尊彝』铭文。以云雷纹为地，上饰兽面纹、夔龙纹、凤鸟纹，下半部是缶字，是盛酒器，多用青铜制造，有圆形和方形两种，皿方罍是迄今出土的方罍中最大最精美的一件，称“方罍之王”。
  • 法国巴黎彤阁。
  • 1956年盖收藏在湖南博物馆。
  • 1992上海博物馆馆长马承源先生在新田东一家见到器身。
  • 2001年美国纽约佳士得春拍，924.6万美元成交被法国买走，中国青铜器拍卖价格之最。

# 第七期：上海博物馆

# 商鞅方升

  • 国宝守护人：黄磊
  • 标准量器，统一度量衡商鞅变法目前为止唯一实物例证。
  • 商鞅方升底部铭文。一段是秦孝公十八年公元前344年，内容关于商鞅变法和统一度量衡；一段是秦始皇二十六年公园前221年。两段铭文跨度百年。
  • 秦穆公重用楚国百里奚，宋国蹇叔，晋国丕豹、公孙支兼并十二国，称霸西戎。
  • 秦孝公重用卫国商鞅，实施变法，国富民强，打败楚魏，扩地千里。
  • 秦惠文王重用卫国张仪，拆散六国合纵，转为连横。
  • 秦昭王重用魏国范雎，远交近攻，蚕食诸侯。
  • 秦始皇，重用楚国李斯，扫六合，成帝业。
  • 公园前221年，秦始皇下令全国“车同轨，书同文”，中华民族第一次实现真正意义上统一。
  • 最真实的安全感来自最真实的计量数据。
  • 量子计量
  • 伏羲女娲图，1965年出土于新疆阿斯塔纳76号墓，唐代绢画，画左女娲执规，画右伏羲执矩。
  • 以度审容，以长度描述容量。

# 朱克柔缂丝莲塘乳鸭图

  • 国宝守护人：那英
  • 南宋朱克柔的传世之作，是迄今现存宋代缂(ke)丝作品中最大的杰作。
  • 朱刚，字克柔，女华亭县(今上海松江)人。
  • 南宋高总，江东丝织重镇松江，江南大丝商庞家儿媳。
  • 乳鸭并排，白鹭成双，朱刚南宋缂丝第一织手。
  • 长短戗，缂丝戗(qiang)色技法之一。蚕丝是一种蛋白质，紫外线会对它有伤害。
  • 1972年香港实业家霍英东的女儿霍丽娜。希望能复刻“莲塘乳鸭图”。
  • 当代“朱克柔”马惠娟，二十岁开始缂丝，做了四十多年。06年又复制的“莲塘乳鸭图”。
  • 缂丝里的“通经断纬”，正反两面图案一模一样。
  • 中国丝绸之路

# 大克鼎

  • 国宝守护人：易烊千玺
  • 清朝光绪年间出土于陕西扶风县窖藏，鼎内铭文二百九十字，记载膳夫克对祖父功绩的追述和周王对克的赏赐。
  • 该鼎是青铜转变期的代表，和“大盂鼎”“毛公鼎”并称海内三宝。
  • 牢礼之法，周礼中饮食礼制的法纲。
  • 出纳王命，向百姓传达王命，也向王反映民情。
  • 清朝末年，清代著名金石收藏大家潘祖荫集齐西周两个最大的青铜器，一件大盂鼎，一件大克鼎。潘达于守护大克鼎，保护它度过战乱的浩劫，建国后主动将大克鼎捐赠给国家，后陆续捐赠家中收藏的珍贵文物。
  • 今生守护人，潘达于奶奶的长孙潘裕翼，潘祖荫的第五代重孙。
  • 海内三宝(清朝挖掘的三件青铜器)，潘有其二(大盂鼎，大克鼎)。毛公鼎在台北博物馆(五百字铭文)，大盂鼎在国家博物馆(三百字铭文)，大克鼎上海博物馆(二百九十字铭文)。
  • 三百八十件青铜器，一件一张照片。
