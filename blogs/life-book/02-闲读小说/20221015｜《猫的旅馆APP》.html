# 《猫的旅馆APP》孤独症女主 & 流浪猫 & 治愈

# 简述

作为一个资深的小说迷，也算阅“小说”无数，乍读本书觉得立意不错可以一读，抱着试试的态度，越读越上头，想看小女主什么时候可以说话和人正常交流，想看猫咪们的日常（以喵咪的角度写对铲屎官的吐槽😂）。

本书亮点：

  • 立意很好，讲述孤独症女主和流浪猫之间相互治愈的温馨日常故事；
  • 感情线，没有常见小说的通病给小小年纪的女主安排感情线，但讲述了亲情、友情，还有人与动物之间相互治愈的美好情感；
  • 故事叙述从开始的轻松悠闲，到后来的感动落泪，有笑点有泪点；
  • 人物和喵咪性格也各有特点。

# 简介

夏安安是位孤独症儿童，四岁半那年刚随妈妈搬去新的住处，她的世界突然多了一款名叫 “猫的旅馆” 的APP。每年冬天幸福小区周边的流浪猫都会因为寒冷和饥饿折损不少成员，今年自从夏安安和妈妈搬来以后，它们有吃有住，来年春天，猫群不但全部存活，还多了几窝小猫崽。 猫群决定将夏安安列为重点守护对象，陪伴她成长。

在 “猫的旅馆APP” 帮助下，夏安安帮助、爱护、救助流浪猫和流浪狗，逐渐克服心理阴影不再孤独，还交了很多新朋友。

# 作者

兔耳齐，晋江作者

# 人物

夏安安：小女主，因家庭原因患有孤独症不爱说话，专注，画画和音乐天赋，有爱心；

夏诗桔：女主妈妈，为了女主搬离城市，爱护女儿，有耐心，支持女儿的决定；

周言天：从小点亮“社交牛逼症”天赋，是帮助安安救助猫咪、走出孤独的重要朋友之一；

季宥宇：很喜欢小猫咪的钢琴演奏家，安安和周言天的钢琴老师，收养了猫咪莱茵；

大人物：任雯（夏诗桔闺蜜，安安的阿姨）、杜医生（安安的主治心理医生）、姜薇（国家运动员因车祸残疾，收养了叫妞妞的狗狗，一起参加飞盘赛）；

小伙伴：汤一萌、艾米、席小雪、于洁、赵小妮、徐以昭

# 猫咪 🐈

多崽：橘猫，能听懂人类语言，是帮助安安救助猫咪、走出孤独的重要朋友之一；

老鹰：流浪猫群的统领；

初八：傲娇高贵又口是心非的家养猫，喜欢参与流浪猫群的会议；

其他：莱茵、花花、大吉、元宝、刚出生的小猫崽若干

# 狗狗 🐶

# 瞬间

# 幼儿园的矛盾场景

犯了错的小朋友在家长的教导下会认识自己的错误并道歉。

# 某明星和子女场景

女儿因富养会娇气有些小任性，但善良会随着环境的改变而学习成长，不会有常见小说的趾高气扬。

# 安安倾诉打开心扉

安安为了帮猫咪找归属向心理医生倾诉内心深处的秘密，边哭泣边讲述幼时记忆里父亲、奶奶的嫌弃、谩骂，言语的伤害致其失言封闭自己，读到这里时一度气愤父亲的无情、奶奶的重男轻女，心疼安安和妈妈遭遇，甚至于落泪。

# 猫猫们的过往

每一条流浪猫都是有故事的猫咪，无意间的走失、各种理由的抛弃、恶意的伤害，穿梭在城市间、街道旁、灌木里的身影弱小却又强大。
