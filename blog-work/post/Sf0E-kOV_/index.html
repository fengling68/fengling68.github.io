vuedraggable拖拽库

2019-07-03 vue 插件

一、安装

npm i -S vuedraggable

二、引入

import draggable from 'vuedraggable';

components: {
  draggable,
},

三、使用(操作数据)

data() {
  return {
    myArray1: [1, 2, 3, 4, 5],
    myArray2: [6, 7, 8, 9, 10],
  };
},

<div>
  <draggable
    group="people"
    v-model="myArray1"
    @choose='onChoose1'
    @clone='onClone1'
    @start='onStart1'
    @update='onUpdate1'
    @change="onChange1"
    @add='onAdd1'
    @remove='onRemove1'
    @sort='onSort1'
    @end='onEnd1'
    @filter='onFilter1'
  >
    <p
      v-for="(item, i) in myArray1" v-bind:key="i"
    >draggableOne---{{item}}</p>

    <button slot="header" @click="addEle">draggableOne---Add</button>
    <button slot="footer" @click="removeEle">draggableOne---Remove</button>
  </draggable>

  <draggable
    group="people"

</div>

四、注意事项

draggable：
    v-model="myArray" 或者 :list="myArray"
    draggable=".item" 表示class="item"的元素可拖动
    dhandle=".handle" 表示class="handle"的元素可控制拖动
    group="samegroup" 如要两个以上容器间拖动添加相同的group名

draggable子元素上：
    :disabled="false" 禁止移动

其他：
    点击删除元素时只触发choose事件
    添加删除元素用可以改变原数组和对象的方法
