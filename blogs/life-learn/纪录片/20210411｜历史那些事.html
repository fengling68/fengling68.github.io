  • 后九鼎沉于河

  • 魏晋清谈，重思辨，重逻辑，重审美。
  • 五石散配方：丹砂、雄黄、白矾、曾青、磁石。
  • 砒霜，砷元素。
  • 何宴被司马懿诛灭三族
  • 爱美、空谈、嗑药，何宴。
  • 嵇康，古代十大美男子之一。

# 原来你也是“猫奴”

  • 五千年前，巴比伦河流域，野生斑猫，自我驯化。
  • 巴比伦 -> 埃及 -> 欧洲 -> 东方(两千年前)
  • 唐代上层养猫兴盛，唐代仕女图。
  • 狸奴、衔蝉都是对猫的称呼
  • 黄庭坚、陆游、明嘉靖皇帝朱厚熜、清朝张之洞。
  • 武则天禁猫
  • 故宫博物院生活了两百多只猫
  • 明代设猫儿房
  • 明朱瞻基作《花下狸奴图轴》
  • 明孝宗朱佑樘在乾明门养了十一只猫
  • 朱厚熜、严嵩、虬龙碑
  • 公元 1232 年，罗马教皇格里高利九世发布教谕《罗马之声》，把黑猫和撒旦联系一起，抓捕处死黑猫，公元 1347 至 1353 年席卷欧罗巴的鼠疫大瘟疫，夺走了两千五百万欧洲人的性命，占当时欧洲人口的三分之一。

# 戏精的诞生

  • 越王勾践剑，吴王夫差矛，春秋乱世。
  • 公园前 494 年，越王勾践被围，后为奴侍奉吴王夫差。尝屎问吉。后吴王夫差放勾践回国。
  • 西汉末年，官二代王获打死了丫鬟，父亲王莽逼儿子自杀，人设太完美。后窃取皇权。十五年后因不合时宜的新政被赶下皇位。
  • 唐光王李忱，结束牛李党争，大中之治。
  • 唐朝皇宫大明宫

# 赢在大明

  • 公元 1591，明朝余象斗，落榜后转书商，卖教辅书。
  • 评三国水浒，图文评三位一体。当代人的营销手段当年余象斗都玩过的。
  • 评林本，后出现《西游记》，蹭热度，加本人防伪肖像水印。
  • 后出精装版，典藏版，双峰堂仰止大叔八大奇书。
  • 余象斗出版界营销大师

# 百变巨咖秀

  • 清王朝爱新觉罗胤禛雍正皇帝
  • 铁腕治国，大力改革，严惩贪污腐败
  • 一年只休息几天
  • 《雍正行乐图》扮演古名人、农夫、外国洋装、道士僧侣等等
  • 郎世宁画画
  • 公元 1700 年法国凡尔赛宫波旁王朝国王路易十四，主办化妆舞会，穿龙袍。
  • 神父白晋前往过清朝，路易十四和康熙互通书信。
  • 公元 1735 年，五十七岁雍正驾崩。
  • 乾隆也cosplay，慈禧也扮演过观音。

# 大唐依然在

  • 唐贞观二十三年，公元 649 年唐太宗李世民驾崩。
  • 异族人愿殉葬，被唐高宗按太宗遗召驳回。
  • 公元 626 年，被迫签订渭水之盟。
  • 公元 632 年可汗契芯何力，带族人归顺，封左领军将军，驻守大唐门户甘凉二州。
  • 公元 636 年都布可汗阿史那社尔归顺唐朝，正三品左骁卫大将军，逝世后得到陪葬昭陵的殊荣。
  • 唐天宝年间，胡旋女。
  • 公元八世纪长安街，富甲一方垄断丝路的“昭武九姓”粟特人、请求帮助复国的波斯人、传播佛法天竺人、接受雇佣担任蕃将的回鹘人、日本人、新罗人、高句丽人。
  • 唐代最大版图占亚洲四分之一
  • 唐长安城世界城市中心
  • 盗月社、联动、美食
  • 唐天宝十二年，公元 753 年，李白《哭晁卿衡》，但晁衡并未死。晁衡遣唐使。
  • 白江口之战
  • 公元 907 年唐灭
  • 大唐帝国，唐人街
  • 汉国家内部的自我认同，唐来自外部的肯定。代表了泱泱中华的国际影响力，新中国将是下一个巅峰。
