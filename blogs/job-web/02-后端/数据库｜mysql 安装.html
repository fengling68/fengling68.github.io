# mysql 安装

# 下载安装

官网下载地址：https://dev.mysql.com/downloads/mysql/ 可视化图形工具：SQLPro Studio

当弹出一个MYSQL Installer提示框的时候一定打开备忘录复制粘贴记下弹出框的密码

系统偏好设置 ——> MySql ——> start mysql server

# 环境变量

打开：vim ~/.bash_profile 添加：PATH=$PATH:/usr/local/mysql/bin 命令：source ~/.bash_profile

# 登录 sql

mysql -u root -p

# 常用命令

CREATE DATABASE new_database;
show databases;
use databases
show tables

# 修改强密码

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'password'; (更新用户密码 )
