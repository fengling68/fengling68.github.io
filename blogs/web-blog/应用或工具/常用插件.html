# 常用插件

# Chrome 插件

  • User-Agent Switcher（随时切换浏览器的 UA(浏览器标识)，一键模拟各种浏览器和手机浏览器，开发者必备助手）
  • Proxy SwitchyOmega（浏览器代理设置神器）
  • Click and Clean（清除浏览器缓存、历史记录等）
  • Undo Close Tab（恢复最近关闭的标签页）
  • ImageAssistant（批量下载网页图片的神器）
  • Clutter Free（关闭重复打开的标签页）
  • Tampermonkey（安装管理油猴脚本）
  • Cache Killer（加载网页前自动清除缓存）
  • AdblockPlus（拦截广告插件）
  • VueDevTools（vue 开发扩展工具）
  • Wappalyzer（分析网站使用技术插件）
  • ARC Welder（谷歌官方推出的安卓模拟器）
  • FEHeapler（前端常用小工具，如 json 格式化、时间戳、正则等）
  • Convertio（文档、音频、视频、图片格式转换）
  • fireShot（chrome 滚动截屏插件）
  • InFinity（个性化 tab 页）
  • OneTab（整理打开的网页）
  • IETab（切换到 IE 浏览器内核）
  • Eagle（图片收藏管理插件）
  • 掘金（掘金文章插件）
  • 彩云小译（中英文对照翻译）
